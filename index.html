Chatter

Solo Project

Date Created: August 2025

Link to repository

Description: Twitch chat client build using Electron, Vite, TypeScript, and React.

This project was something I was interested in doing for a while, but I just never knew how to approach it. It had always felt a bit overwhelming since I had never created a desktop app. However, after learning about Electron, I realized I could just use my web development skills to create a desktop app.

Not only did this project teach me about Electron and how it works, I also had to dive deep into the Twitch API in order to make it work. Some of the challenges this project brought were:

  • Properly connecting to the Twitch EventSub WebSocket server in order to get chats from the correct channel
  • Parsing messages in order to display emotes and mentions
  • Adding real time text to image conversion in the message area while typing
